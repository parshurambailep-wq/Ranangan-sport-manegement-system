## 📌 Project Overview

**Ranangan** is a full-stack Sports Event Management System developed to simplify and digitize the process of organizing sports tournaments in colleges, universities, and organizations.

This platform provides a centralized solution for managing teams, players, matches, event scheduling, ticket bookings, and payments — reducing manual effort and improving coordination between organizers and participants.

---

* ✅ Team & Player Registration
* ✅ Tournament Creation
* ✅ Match Scheduling
* ✅ Admin Dashboard
* ✅ Event & Venue Management
* ✅ Ticket Booking System
* ✅ Role-Based Login (Admin / Organizer / User)
* ✅ Real-Time Score Updates
* ✅ UPI Payment Transaction ID Input

* ✅ Secure Authentication
